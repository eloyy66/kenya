Para instalar el repo usa:
git clone https://github.com/eloyy66/kenya

te vas a la carpeta del proyecto:
cd kenya

creas un entorno virtual de python con las librerías requeridas:
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
**(después de esto, te debe aparecer algo como "(venv) [tu_usuario]%"**
