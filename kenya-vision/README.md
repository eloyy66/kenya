Para instalar el repo usa:
git clone https://github.com/eloyy66/kenya

te vas a la carpeta del proyecto:
cd kenya

creas un entorno virtual de python con las librerías requeridas:
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
**(después de esto, te debe aparecer algo como "(venv) [tu_usuario]%"**


es preferible que los códigos (como prueba.py o main.py) se ejecuten desde la terminal:
python --version (de preferencia 3.10.xx)
o python3 --version
python archivo.py

desactiva el venv cuando termines de chambear:
deactivate
